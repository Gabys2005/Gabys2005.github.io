Part Counter

A very simple plugin that allows you to see the total part count in your game, workspace or selection. Has the ability to show you individual counts for bricks, meshes and unions, as well as displaying you a list of all meshes in the game.

Texture Placer

This plugin allows you to quickly place the same texture on multiple surfaces, so you don't have to duplicate an existing texture or make a new one for each of them manually.
