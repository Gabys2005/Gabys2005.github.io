Camera System v4 API


Getting access to the API

To start using the API, simply require it, it gets added to the main CameraSystem folder when the game starts 

local api = require(workspace.CameraSystem:WaitForChild("API"))

api:GetCamIdFromName()

GetCamIdFromName(type, name) - returns camera id

local id = api:GetCamIdFromName("Moving", "Example")

                print(id) => 1

api:ChangeCam()

ChangeCam(type, id) - returns nothing, changes the current camera

api:ChangeCam("Moving", 1)

api:GetCurrentCam()

GetCurrentCam() - returns a table with the type, id, name and time (if current cam is a moving cam) and the camera model 

local cam = api:GetCurrentCam()

                print(cam) => {Type = "Moving", Id = 1, Name = "Example", Time = 5, Model = workspace.CameraSystem.Cameras.Moving.Example}

api:Focus()

Focus(playerText) - attempts to focus on the player with username starting with playerText, if playerText is nil or "" then it will remove any focus, returns the focused player or nil 

local focused = api:Focus("gab")

                print(focused.Name) => "gabys2005"

api:GetFocus()

GetFocus() - returns currently focused player or nil

local focused = api:GetFocus()

                print(focused.Name) => "gabys2005"

api:Blur()

Blur(level, time) - sets the blur to the specified level in the specified time, time defaults to 0.1 

api:Blur(10)

api:GetBlur()

GetBlur() - returns the current level of blur

local blur = api:GetBlur()

                print(blur) => 10

api:Fov()

Fov(level, time) - sets the FOV to the specified level in the specified time, time defaults to 0.1 

api:Fov(50)

api:GetFov()

GetFov() - returns the current FOV

local fov = api:GetFov()

                print(fov) => 50

api:Blackout()

Blackout(enabled, color) - enables or disabled the blackout, if color is specified then sets the blackout color to color 

api:Blackout(true, Color3.fromRGB(0,255,0))

api:GetBlackout()

GetBlackout() - returns a bool representing if the blackout is active, and a color of the current blackout 

local enabled, color = api:GetBlackout()

                print(enabled, color) => true, 0,1,0

api:Saturation()

Fov(level, time) - sets the saturation to the specified level in the specified time, time defaults to 0.1 

api:Saturation(-1)

api:GetSaturation()

GetSaturation() - returns the current saturation value

local saturation = api:GetSaturation()

                print(saturation) => -1

api:TintColor()

TintColor(color, time) - sets the tint color to the specified color in the specified time, time defaults to 0.1 

api:TintColor(Color3.fromRGB(0,255,0))

api:GetTintColor()

GetTintColor() - returns the current tint color

local color = api:GetTintColor()

                print(color) => 0,1,0

api:Bars()

Bars(enabled) - enables or disables the bars

api:Bars(true)

api:GetBars()

GetBars() - returns a bool indicating if the bars are enabled or not

local enabled = api:GetBars()

                print(enabled) => true

api:BarSize()

BarSize(size) - sets the size of one bar to take up this much screen (for example size 25 will make 1 bar take up 25% of the screen) 

api:BarSize(25)

api:TransitionFade()

TransitionFade(enabled, color) - enables or disables the transition between 2 cameras, if color is defined then sets the transition color to color 

api:TransitionFade(false)

api:GetTransition()

GetTransition() - returns a bool indicating if the transition is enabled or not, and the color of the transition 

local enabled, color = api:GetTransition()

                print(enabled, color) => false, 0,0,0

api:AskForValues()

AskForValues(plr, data) - used by macros, ask the user (plr) for values
Data is a table, where the first thing is the name that you want visible on the gui.
All things after that are the values you want, every value is a table with Name and Type.
Type can be set to String, Number or Slider. If Type is Slider then you also need to specify Min and Max values for it
returns a tuple (multiple values), values that you requested, if the first value is nil then the user cancelled the prompt 

local fov, exampleString, exampleNum = api:AskForValues(plr, {"API Example", {Type = "Slider", Name = "SliderExample", Min = 0, Max = 100}, {Type = "String", Name = "StringExample"}, {Type = "Number", Name = "NumberExample"}})

                if fov then

                    print(fov, exampleString, exampleNum)

                else

                    print("User cancelled the prompt")

                end

api:On()

On(eventName, func) - allows you to run your own code whenever something happens in the system (like a camera change). Below are all of the possible events with the parameters they pass on (name: params)

CamChange: camType,camId

FocusChange: player/nil - will return the player if found any, nil if none were found or unfocused

BlurChange: level,time

FovChange: fov,time

Blackout: visible,color - color is optional

SaturationChange: level,time

BarsToggle: visible - when the bars are being turned on or off

BarSizeChange: size - when the size of the bars is being changed

TransitionModeChange: mode

api:On("CamChange", function()

                local currentCam = api:GetCurrentCam()

                print("Changed the camera to: " .. currentCam.Name)

            end)
