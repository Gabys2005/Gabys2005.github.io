Basic usage

The module has 2 functions - newGIF() and EndAllGifsIn() (described in the next section),

Screen = a frame where the gif should be1,
Image = ID of a spritesheet2,
FPS = fps (up to 60)3,
FramesPerRow = how many frames are in 1 row,
Frames = total number of frames

1 screen needs to point to a SurfaceGui, or any gui object like Frame, Scrolling Frame, TextLabel...
2 The Image needs to be a spritesheet. Click here to convert your GIF into a sprite sheet
3 The GIF may be slower depending on the server lag created in the game
Note: in the example below the module is in ServerScriptService

local GIFPlayer = require(game.ServerScriptService.GIFPlayer)

                

local gif = GIFPlayer.newGIF({Screen = workspace.Screen.SurfaceGui.GIF,

                            Image = "rbxassetid://3528512681",

                            FPS = 24,

                            FramesPerRow = 6,

                            Frames = 26})

                                            
gif:Play()

All functions

gif:Play() - plays the GIF

gif:Stop() - stops the GIF

gif:Reset() - resets the GIF (puts it on the 1st frame)

gif:End() - removes the GIF


Stopping all GIFs in an instance

EndAllGifsIn(Instance) is a function that runs gif:End() on all GIFs that are a descendant of a location put as a variable

local GIFPlayer = require(game.ServerScriptService.GIFPlayer)
    


GIFPlayer.EndAllGifsIn(workspace.Prop)

Using multiple spritesheets

The code below shows how to use multiple spritesheets in the same GIF (for better quality)

Note: you can add more than 2 sheets

local GIFPlayer = require(game.ServerScriptService.GIFPlayer)
            


local sheet1 = {Image = "rbxassetid://5138441857",

                FPS = 2,

                FramesPerRow = 4,

                Frames = 20

                }

                        

local sheet2 = {Image = "rbxassetid://3528512681",

                FPS = 24,

                FramesPerRow = 6,

                Frames = 26

                }

                        

local gif = GIFPlayer.newGIF({Screen = workspace.Screen.SurfaceGui.GIF,

                            Mode = "Advanced",

                            SpriteSheets = {sheet1, sheet2})

                        
     
gif:Play()
