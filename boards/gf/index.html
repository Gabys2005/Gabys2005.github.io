Some info

The "Sync Televoting" button only works if you're using the televoting created by me (available here)
You can change the colors and stuff like that just by editing the properties of Frames (the point and country templates are inside of the "Main" script)
The "Countries" frame has a UIGridLayout in it, but it's deleted when the game starts (you still need to set the size of both the CountryTemplate and UIGridLayout to the same value)
