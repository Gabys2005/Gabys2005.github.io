Installation

Click the button below to get redirected to Roblox's library page. Take the asset from there and use the Toolbox in Studio to insert it into your game.

Roblox Library

Basic usage

Whenever someone joins the game, their name will show up on the panel. You can click on it to make their bubble chat range infinite. You can duplicate the included panel if you want to be able to control it from multiple places.


Zones

You can place zones around your game in places where you want every player to have infinite bubble chat range (for example a stage). If you don't want to be annoyed with the big blocks in Studio, you can move the "LongBubbleChatZones" folder into ReplicatedStorage
