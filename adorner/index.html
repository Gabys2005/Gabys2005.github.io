Screen Adorner

What is a screen adorner?

Screen adorner allows you to display 1 frame on multiple parts while using SurfaceGuis, it's mostly used for GIFs


Download

Click the button below to get redirected to Roblox's library page. Take the asset from there and use the Toolbox in Studio to insert it into your game.
You can also just copy the code from Github Gist and paste it into a ModuleScript

Download Github Gist

Usage

The module has only 1 function - adorner:Adorn()

The functions requires 3 parameters:

- Frame - a frame to replicate on all the screens
- Screens - a table of SurfaceGuis the frame should be on
- TopMarker - a Vector3 position of the very top of the screens (to calculate the height offset, just put it as part.Position)

The Frame's size needs to be specified in pixels, if you use default SurfaceGui properties, then the size of the frame needs to be 50 times bigger than the size of the screens

The SurfaceGuis need a XOffset value with the X (left-right) offset of that screen in pixels 

local adorner = require(game.ServerScriptService.ScreenAdorner)

                

local surfaces = {workspace.Screen1.SurfaceGui,

                  workspace.Screen2.SurfaceGui}

local marker = workspace.Marker.Position



adorner:Adorn(script.Frame, surfaces, marker)

Removing the frames

The Adorn() function returns an object with a :Destroy() function which will remove the frame from all screens 

... // The stuff from above
    

local adorned = adorner:Adorn(script.Frame, surfaces, marker)

wait(10)

// Remove the frames

adorned:Destroy()
